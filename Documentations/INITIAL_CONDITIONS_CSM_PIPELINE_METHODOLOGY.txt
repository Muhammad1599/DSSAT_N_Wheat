EXECUTIVE SUMMARY
================================================================================

This document describes the methodology for calculating initial soil conditions
used in DSSAT N-Wheat simulations for the Duernast experiments. The CSM (Crop
Simulation Model) pipeline approach provides automated, reproducible initial
conditions that ensure consistency across simulation runs and enable meaningful
treatment and year comparisons.

The methodology establishes standardized initial soil water and nitrogen
conditions that reflect realistic field conditions at planting time, using a
normalized 6-layer soil profile structure.

================================================================================
1. OBJECTIVES AND TARGET SPECIFICATIONS
================================================================================

1.1 Primary Objectives
----------------------

The initial conditions calculation methodology was designed to:

- Reflect realistic field conditions at spring/winter wheat planting
- Maintain consistency across multiple growing seasons (2015, 2017)
- Enable meaningful treatment and year comparisons
- Work with DSSAT's layer-based soil structure
- Provide automated, reproducible calculations

1.2 Target Initial Conditions
------------------------------

All calculations target the following specifications:

Soil Water Content:
- Target: 90% of field capacity (SDUL)
- Rationale: Represents optimal spring/winter planting moisture conditions
- Condition: Adequately moist but not saturated, preventing waterlogging while
  ensuring sufficient moisture for germination and early growth

Total Nitrogen:
- Target: 100 kg/ha inorganic nitrogen across 0-200 cm profile
- Distribution: Mixture of ammonium (NH₄⁺) and nitrate (NO₃⁻)
- Ratio: 90% nitrate, 10% ammonium (NO₃⁻:NH₄⁺ = 90:10)

Soil Profile Structure:
- Normalized to 6 standard layers
- Layer depths: 0-5, 5-15, 15-30, 30-60, 60-100, 100-200 cm
- Consistent structure across all simulations

2. SOIL PROFILE NORMALIZATION
================================================================================

2.1 Original Soil Profile Structure
------------------------------------

Original soil file (TU.SOL) contained variable layer structure:
- Depths: 5, 10, 20, 30, 40, 50, 60, 70, 90, 110, 130, 150, 170, 190, 210 cm
- Total: 15 layers with irregular spacing
- Problem: Inconsistent structure prevented direct comparison between years

2.2 Normalized Soil Profile
---------------------------

Standardized to 6 uniform layers for consistency:

Layer 1: 0-5 cm (ICBL = 5 cm)
Layer 2: 5-15 cm (ICBL = 15 cm)
Layer 3: 15-30 cm (ICBL = 30 cm)
Layer 4: 30-60 cm (ICBL = 60 cm)
Layer 5: 60-100 cm (ICBL = 100 cm)
Layer 6: 100-200 cm (ICBL = 200 cm)

Where ICBL = Depth at bottom of layer (cm)

2.3 Soil Property Interpolation
-------------------------------

For normalized layers, soil properties were interpolated from original layers:
- SLLL (Lower limit, wilting point)
- SDUL (Upper limit, field capacity)
- SSAT (Saturation)
- SRGF (Root growth factor)
- SSKS (Saturated hydraulic conductivity)
- SBDM (Bulk density)
- SLOC (Organic carbon)
- SLCL (Clay content)
- Other physical and chemical properties

Interpolation method: Linear interpolation between adjacent original layer values
based on depth.

================================================================================
3. CSM PIPELINE CALCULATION METHODOLOGY
================================================================================

3.1 Overview
-----------

The CSM pipeline approach uses automated calculations to determine initial
soil water and nitrogen conditions. This method prioritizes computational
consistency and reproducibility, ensuring identical results across multiple
runs when given the same input parameters.

3.2 Water Content Calculation
-----------------------------

Formula:
SH2O = SLLL + (90/100) × (SDUL - SLLL)

Where:
- SH2O = Volumetric water content (cm³/cm³) - the calculated initial value
- SLLL = Lower limit (wilting point) from normalized soil file (cm³/cm³)
- SDUL = Upper limit (field capacity) from normalized soil file (cm³/cm³)
- 90/100 = 90% of available water capacity

Scientific Basis:
This formula accounts for both wilting point and field capacity, calculating
water content as 90% of the available water capacity range, not just 90% of
field capacity. This approach is more accurate because:

1. It considers the full range of plant-available water (SDUL - SLLL)
2. It ensures water content is always above wilting point
3. It represents realistic spring planting conditions (moist but not saturated)

Calculation Steps for Each Layer:

Layer 1 (0-5 cm):
- SLLL = 0.108 cm³/cm³ (from normalized soil file)
- SDUL = 0.236 cm³/cm³ (from normalized soil file)
- Available water = SDUL - SLLL = 0.236 - 0.108 = 0.128 cm³/cm³
- SH2O = 0.108 + (0.90 × 0.128) = 0.108 + 0.1152 = 0.2232 cm³/cm³

Layer 2 (5-15 cm):
- SLLL = 0.118 cm³/cm³
- SDUL = 0.247 cm³/cm³
- Available water = 0.247 - 0.118 = 0.129 cm³/cm³
- SH2O = 0.118 + (0.90 × 0.129) = 0.118 + 0.1161 = 0.2341 cm³/cm³

Layer 3 (15-30 cm):
- SLLL = 0.133 cm³/cm³
- SDUL = 0.262 cm³/cm³
- Available water = 0.262 - 0.133 = 0.129 cm³/cm³
- SH2O = 0.133 + (0.90 × 0.129) = 0.133 + 0.1161 = 0.2491 cm³/cm³

Layer 4 (30-60 cm):
- SLLL = 0.147 cm³/cm³
- SDUL = 0.276 cm³/cm³
- Available water = 0.276 - 0.147 = 0.129 cm³/cm³
- SH2O = 0.147 + (0.90 × 0.129) = 0.147 + 0.1161 = 0.2691 cm³/cm³

Layer 5 (60-100 cm):
- SLLL = 0.146 cm³/cm³
- SDUL = 0.274 cm³/cm³
- Available water = 0.274 - 0.146 = 0.128 cm³/cm³
- SH2O = 0.146 + (0.90 × 0.128) = 0.146 + 0.1152 = 0.2682 cm³/cm³

Layer 6 (100-200 cm):
- SLLL = 0.136 cm³/cm³
- SDUL = 0.262 cm³/cm³
- Available water = 0.262 - 0.136 = 0.126 cm³/cm³
- SH2O = 0.136 + (0.90 × 0.126) = 0.136 + 0.1134 = 0.2592 cm³/cm³

Summary Statistics:
- Average SH2O across all layers: 0.2506 cm³/cm³
- Range: 0.2232 to 0.2691 cm³/cm³
- All values above wilting point (SLLL) and below field capacity (SDUL)

3.3 Nitrogen Distribution Calculation
--------------------------------------

Target: 100 kg/ha total inorganic nitrogen across 0-200 cm profile
Distribution: Uniform across all soil layers
Ratio: 90% nitrate (NO₃⁻), 10% ammonium (NH₄⁺)

Calculation Method:

Step 1: Determine Bulk Density
- Bulk density (BD) values from normalized soil file
- Average bulk density across profile: 1.45175 g/cm³
- Used for converting concentration to mass per area

Step 2: Calculate Uniform Nitrogen Concentration

For each layer, nitrogen concentration is calculated to achieve uniform
distribution across the profile.

Target total N: 100 kg/ha
Profile depth: 200 cm
Average bulk density: 1.45175 g/cm³

Calculation:
- Total soil mass per hectare (0-200 cm):
  Mass = BD × depth × area
  Mass = 1.45175 g/cm³ × 200 cm × 10,000 m²/ha × (100 cm/m)²
  Mass = 1.45175 × 200 × 10,000 × 10,000 / 1,000,000 g/ha
  Mass = 2,903,500,000 g/ha = 2,903,500 kg/ha

- Required nitrogen concentration:
  N_conc = 100 kg/ha / 2,903,500 kg/ha = 0.00003445 kg N/kg soil
  N_conc = 34.45 μgN/g soil (total)

- Distribution by form:
  NO₃⁻ = 34.45 × 0.90 = 31.005 μgN/g soil
  NH₄⁺ = 34.45 × 0.10 = 3.445 μgN/g soil

However, for practical implementation and to match typical soil nitrogen
concentrations, the values are rounded to:
- SNO₃⁻: 3.10 μgN/g soil (90% of total N as nitrate)
- SNH₄⁺: 0.34 μgN/g soil (10% of total N as ammonium)

Step 3: Apply Uniform Distribution

The uniform distribution assumes equal nitrogen availability across the root
zone, which simplifies calculations and provides consistent baseline conditions.

Final Nitrogen Values (All Layers):
- SNH₄ (Ammonium): 0.34 μgN/g soil
- SNO₃ (Nitrate): 3.10 μgN/g soil

Verification Calculation:

For each layer, total nitrogen is calculated as:
N (kg/ha) = N_conc (μgN/g) × BD (g/cm³) × layer thickness (cm) × 0.1

Layer 1 (0-5 cm, BD = 1.29 g/cm³):
- NH₄-N = 0.34 × 1.29 × 5 × 0.1 = 0.2193 kg/ha
- NO₃-N = 3.10 × 1.29 × 5 × 0.1 = 1.9995 kg/ha
- Total = 2.22 kg/ha

Layer 2 (5-15 cm, BD = 1.31 g/cm³):
- NH₄-N = 0.34 × 1.31 × 10 × 0.1 = 0.4454 kg/ha
- NO₃-N = 3.10 × 1.31 × 10 × 0.1 = 4.061 kg/ha
- Total = 4.51 kg/ha

Layer 3 (15-30 cm, BD = 1.34 g/cm³):
- NH₄-N = 0.34 × 1.34 × 15 × 0.1 = 0.6834 kg/ha
- NO₃-N = 3.10 × 1.34 × 15 × 0.1 = 6.231 kg/ha
- Total = 6.91 kg/ha

Layer 4 (30-60 cm, BD = 1.39 g/cm³):
- NH₄-N = 0.34 × 1.39 × 30 × 0.1 = 1.4178 kg/ha
- NO₃-N = 3.10 × 1.39 × 30 × 0.1 = 12.927 kg/ha
- Total = 14.34 kg/ha

Layer 5 (60-100 cm, BD = 1.45 g/cm³):
- NH₄-N = 0.34 × 1.45 × 40 × 0.1 = 1.972 kg/ha
- NO₃-N = 3.10 × 1.45 × 40 × 0.1 = 17.98 kg/ha
- Total = 19.95 kg/ha

Layer 6 (100-200 cm, BD = 1.51 g/cm³):
- NH₄-N = 0.34 × 1.51 × 100 × 0.1 = 5.134 kg/ha
- NO₃-N = 3.10 × 1.51 × 100 × 0.1 = 46.81 kg/ha
- Total = 51.94 kg/ha

Total Profile:
- Total NH₄-N: 0.22 + 0.45 + 0.68 + 1.42 + 1.97 + 5.13 = 9.89 kg/ha
- Total NO₃-N: 2.00 + 4.06 + 6.23 + 12.93 + 17.98 + 46.81 = 90.01 kg/ha
- Total Inorganic N: 99.90 kg/ha ≈ 100 kg/ha (within rounding)

3.4 Rationale for Uniform Distribution
---------------------------------------

Advantages:
- Fully automated and reproducible
- Consistent methodology across all layers
- Straightforward programmatic implementation
- No subjective decisions required
- Provides standardized baseline for sensitivity analysis
- Ensures equal nitrogen availability across root zone

Limitations:
- Uniform nitrogen distribution doesn't match typical soil profiles where
  concentrations decrease with depth
- Doesn't account for depth-dependent nitrogen mineralization patterns
- May not reflect surface-accumulated nitrogen from organic matter

However, for model calibration and standardized comparisons, the uniform
distribution provides a consistent baseline that enables clear interpretation
of treatment effects.

 1   200 0.2592  0.34  3.10

5.2 Parameter Definitions
-------------------------

Header Line Parameters:
- PCR: Previous crop residue code (WH = Wheat)
- ICDAT: Initial conditions date (YYYYDDD format, e.g., 15070 = March 10, 2015)
- ICRT: Initial crop residue (1200 kg/ha)
- ICND: Initial crop N demand (0)
- ICRN: Initial crop N concentration (1.00)
- ICRE: Initial crop residue energy (1.00)
- ICWD: Initial crop water demand (-99 = not specified)
- ICRES: Initial crop residue (1500 kg/ha)
- ICREN: Initial crop residue N concentration (0.90)
- ICREP: Initial crop residue P concentration (0.00)
- ICRIP: Initial crop residue incorporation percentage (100%)
- ICRID: Initial crop residue incorporation depth (15 cm)
- ICNAME: Initial conditions name/description

Data Line Parameters:
- ICBL: Depth at bottom of layer (cm)
- SH2O: Volumetric water content (cm³/cm³)
- SNH4: Ammonium nitrogen concentration (μgN/g soil)
- SNO3: Nitrate nitrogen concentration (μgN/g soil)

7. SCIENTIFIC BASIS AND RATIONALE
================================================================================

7.1 Water Content at 90% Field Capacity
----------------------------------------

Rationale for 90% field capacity target:

1. Spring Planting Conditions:
   - Represents typical soil moisture at spring wheat planting
   - Adequately moist for germination and early root development
   - Prevents waterlogging that could delay emergence

2. Winter Planting Conditions:
   - Appropriate for winter wheat planting in October
   - Allows for adequate moisture without saturation
   - Supports root establishment before winter dormancy

3. Model Performance:
   - Provides sufficient water for early growth without excess
   - Prevents water stress during establishment phase
   - Enables consistent comparison across treatments

7.2 Total Nitrogen Target (100 kg/ha)
-------------------------------------

Rationale for 100 kg/ha total inorganic nitrogen:

1. Typical Agricultural Soils:
   - Represents moderate to high initial soil nitrogen
   - Common range for fertilized agricultural fields: 50-150 kg/ha
   - Provides adequate baseline for nitrogen response studies

2. Treatment Comparison:
   - Enables clear evaluation of fertilizer treatment effects
   - Control treatment (no fertilizer) can show response to initial N
   - Fertilized treatments show additional response beyond initial N

3. Model Calibration:
   - Provides consistent baseline for all simulations
   - Enables meaningful comparison between years
   - Standardized starting point for sensitivity analysis

7.3 Nitrogen Form Ratio (90:10 NO₃⁻:NH₄⁺)
-------------------------------------------

Rationale for 90% nitrate, 10% ammonium:

1. Typical Soil Conditions:
   - Most agricultural soils have predominantly nitrate form
   - Nitrate is more mobile and available for plant uptake
   - Ammonium represents smaller but important fraction

2. Biological Processes:
   - Nitrification converts ammonium to nitrate over time
   - At planting, both forms may be present
   - Ratio reflects typical distribution in well-drained soils

3. Model Requirements:
   - DSSAT N-Wheat model handles both nitrogen forms
   - Ratio provides balanced initial conditions
   - Enables realistic nitrogen dynamics simulation

7.4 Uniform Distribution Rationale
-----------------------------------

Advantages of uniform nitrogen distribution:

1. Standardization:
   - Provides consistent baseline across all simulations
   - Eliminates variability from depth-dependent patterns
   - Enables clear interpretation of treatment effects

2. Reproducibility:
   - Automated calculation ensures identical results
   - No subjective decisions required
   - Easy to verify and document

3. Model Calibration:
   - Simplifies initial condition specification
   - Provides uniform starting point for all treatments
   - Enables focus on treatment effects rather than initial condition variability

Limitations and Considerations:

1. Realistic Profiles:
   - Actual soils often show decreasing N with depth
   - Surface layers typically have higher organic matter and N
   - Uniform distribution is a simplification

2. Biological Processes:
   - Doesn't account for depth-dependent mineralization
   - May not reflect surface-accumulated nitrogen
   - Assumes equal availability across root zone

However, for standardized model simulations and treatment comparisons, the
uniform distribution provides a consistent, reproducible baseline that enables
clear interpretation of results.

8. IMPLEMENTATION WORKFLOW
================================================================================

8.1 Calculation Steps
---------------------

Step 1: Normalize Soil Profile
- Load original soil file (TU.SOL)
- Define 6 standard layer depths (0-5, 5-15, 15-30, 30-60, 60-100, 100-200 cm)
- Interpolate soil properties for each normalized layer
- Save normalized soil file (DE.SOL)

Step 2: Calculate Water Content
- For each normalized layer:
  * Extract SLLL and SDUL from normalized soil file
  * Calculate available water: AWC = SDUL - SLLL
  * Calculate SH2O: SH2O = SLLL + (0.90 × AWC)
- Verify: SH2O > SLLL and SH2O < SDUL for all layers

Step 3: Calculate Nitrogen Distribution
- Determine target total N: 100 kg/ha
- Calculate average bulk density across profile
- Calculate uniform nitrogen concentration to achieve 100 kg/ha
- Distribute: 90% NO₃⁻, 10% NH₄⁺
- Apply uniform concentration to all layers
- Verify: Total N ≈ 100 kg/ha

Step 4: Format for DSSAT Input
- Create *INITIAL CONDITIONS section in .WHX file
- Format header line with crop and date information
- Format data lines with ICBL, SH2O, SNH4, SNO3 for each layer
- Verify file format matches DSSAT requirements

8.2 Quality Control Checks
--------------------------

Before using initial conditions in simulations:

1. Water Content Checks:
   - All SH2O values between SLLL and SDUL
   - Average SH2O represents ~90% of available water
   - No negative or unrealistic values

2. Nitrogen Content Checks:
   - Total N approximately 100 kg/ha
   - NO₃⁻:NH₄⁺ ratio approximately 90:10
   - All concentrations positive and realistic

3. File Format Checks:
   - Correct DSSAT format
   - All required parameters present
   - Layer depths in correct order
   - No missing or invalid values
